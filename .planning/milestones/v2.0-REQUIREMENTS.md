## Notes on Changed Requirements

**R-2.04 (File cache):** Original requirement said "per user" but decision D-01 locked it as global (no user_id in cache key). Same query from any user hits the same cache row — intentional for quota efficiency.

**R-2.11 (Watched state):** Not shipped in v2.0. Watched checkbox persistence remains in localStorage. Deferred to v3.0 backlog.
