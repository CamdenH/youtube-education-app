## Interaction States

Each video's hints UI cycles through these states independently. No other video is affected when one video's state changes.

| State | Visual Description |
|-------|--------------------|
| **Default (no hints fetched)** | `.btn-hints` visible and enabled. `.hint-text` elements hidden. No error shown. |
| **Loading** | `.btn-hints` disabled (`opacity: 0.6; cursor: not-allowed`), text changes to "Loading hints...". No spinner ‚Äî text change is the loading indicator. `.hint-text` elements remain hidden. |
| **Success** | `.btn-hints` hidden (`hidden = true`). All 3 `.hint-text` elements become visible with rendered hint content. Each contains a `.hint-label` span ("Thinking point:") followed by the Socratic nudge text. |
| **Error** | `.btn-hints` re-enabled, text reverts to "Reveal thinking points". A `.hint-error` paragraph appears below the button with error text and `.btn-retry-hint`. |
| **Restored from localStorage** | Identical to Success state ‚Äî button is hidden, hints pre-rendered. Triggered at course render time when `ylc_hints` contains data for this videoId. |

## Interaction Contract

### Trigger Condition
The hint fetch fires on `.btn-hints` click, not on accordion open. The user must expand the questions accordion first (Phase 4 behavior), then explicitly click the button. Source: 05-CONTEXT.md `<specifics>` section.

### Scope Isolation
Each video card's hint state is fully independent. DOM state changes (button disabled, error shown, hints rendered) are scoped to the `.questions-accordion` of the clicked video only. Source: D-03 (CONTEXT.md).

### Atomicity
All 3 hints render together when the `POST /api/hints` response returns. There is no per-hint streaming or progressive reveal. Source: HINT-01, HINT-02 (REQUIREMENTS.md).

### localStorage Persistence Key
`ylc_hints` ‚Äî a flat JSON object keyed by `videoId`, value is an array of 3 hint strings. Written on successful fetch. Read at render time to restore hints without re-fetching. Source: RESEARCH.md Pattern 4 (recommended approach; CONTEXT.md D-01 gives Claude discretion on schema).

### Reload Restore Path
At `renderCourse()` time, after creating `.btn-hints` for each video, check `loadHints(video.videoId)`. If non-null: render hints immediately via `renderHints()`, set `hintsBtn.hidden = true`. Source: RESEARCH.md Pitfall 3.
