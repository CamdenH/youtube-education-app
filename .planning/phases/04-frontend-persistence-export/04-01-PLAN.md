|----------|-------------|

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | index.html rendering | mitigate | All API-sourced strings (video titles, blurbs, questions, channel names) inserted via textContent, never innerHTML — enforced in Plans 02 and 03 |
| T-04-02 | Tampering | localStorage restore | mitigate | Treat localStorage data as untrusted; render via textContent — enforced in Plan 03 |
| T-04-03 | Information Disclosure | markdown export | accept | Export contains only course data user already sees; no secrets or auth tokens involved |
