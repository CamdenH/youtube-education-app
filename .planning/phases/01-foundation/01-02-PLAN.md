**claude.js exports:**

1. `callClaude(fn, ...args)` — async function that:
   - Calls `await fn(...args)`
   - On success, returns the result immediately
   - On error, retries up to 2 more times (3 total attempts)
   - Backoff delays: 1000ms after first failure, 2000ms after second failure (exponential: `1000 * 2^attempt`)
   - Uses `await new Promise(r => setTimeout(r, delay))` for backoff
   - After all retries exhausted, throws the original error
   - Logs retry attempts to console: `console.warn(\`Claude retry \${attempt}/2: \${error.message}\`)`

2. `parseClaudeJSON(text)` — function that:
   - Strips markdown code fences: replace `/^\`\`\`(?:json)?\s*\n?/` at start and `/\n?\s*\`\`\`\s*$/` at end
   - Trims whitespace
   - Calls `JSON.parse()` on the result
   - Returns parsed object
   - Throws SyntaxError if JSON is invalid (no catch — caller handles retry)

Module uses CommonJS: `module.exports = { callClaude, parseClaudeJSON }`.
