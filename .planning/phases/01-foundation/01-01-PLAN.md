  <action>
Create `tests/unit/cache.test.js` first (RED), then implement `cache.js` (GREEN).

**cache.js** must export exactly these functions:
- `cacheGet(filename)` — returns parsed JSON or `null` if file does not exist. Path: `path.join(__dirname, '.cache', filename)`.
- `cacheSet(filename, data)` — writes `JSON.stringify(data, null, 2)` to `.cache/{filename}`. Calls `ensureCacheDir()` first.
- `queryHash(query)` — returns `crypto.createHash('md5').update(query).digest('hex')`.
- `ensureCacheDir()` — creates `.cache/` directory with `{ recursive: true }` if it does not exist.

Use `require('node:crypto')`, `require('node:fs')`, `require('node:path')`.

Cache directory constant: `const CACHE_DIR = path.join(__dirname, '.cache');`

Module uses CommonJS (`module.exports = { cacheGet, cacheSet, queryHash, ensureCacheDir }`).

**Test file** uses `node:test` and `node:assert`. Tests should create a temp `.cache/` directory, run assertions, and clean up. Use `test()` and `assert.strictEqual` / `assert.deepStrictEqual`. Each behavior above is one test case.
