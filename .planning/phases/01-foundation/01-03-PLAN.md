     ```javascript
       part: 'snippet',

       type: 'video',
       maxResults: '8',

     ```

   - On `!res.ok`: call `handleYouTubeError(await res.json())`
   - On success: `cacheSet(cacheKey, data); return data;`

2. `fetchVideoStats(videoIds)` — async function:
   - Check cache for each ID: `const cached = videoIds.map(id => cacheGet(\`video_\${id}.json\`));`
   - Separate into cached hits and uncached IDs
   - If all cached, return immediately (no API call)
   - For uncached IDs, build URL:
     ```javascript
     const params = new URLSearchParams({
       key: process.env.YOUTUBE_API_KEY,
       part: 'snippet,statistics,contentDetails,topicDetails',
       id: uncachedIds.join(',')
     });
     ```
   - Fetch from `https://www.googleapis.com/youtube/v3/videos?${params}`
   - On `!res.ok`: call `handleYouTubeError(await res.json())`
   - Cache each item individually: `items.forEach(item => cacheSet(\`video_\${item.id}.json\`, item));`
   - Return merged array: `[...cachedItems, ...fetchedItems]`

**transcript.js exports:**

1. `parseTimedtextXml(xml)` — function that:
   - Strips all XML/HTML tags: `xml.replace(/<[^>]+>/g, ' ')`
   - Decodes HTML entities:
     ```javascript
     text = text.replace(/&amp;/g, '&')
       .replace(/&lt;/g, '<')
       .replace(/&gt;/g, '>')
       .replace(/&quot;/g, '"')
       .replace(/&#39;/g, "'");
     ```
   - Collapses whitespace: `text.replace(/\s+/g, ' ').trim()`
   - Returns the cleaned string

2. `fetchTranscript(videoId)` — async function:
   - Cache key: `transcript_${videoId}.json`
   - Check cache first: `const cached = cacheGet(cacheKey); if (cached) return cached;`
   - Fetch from `https://www.youtube.com/api/timedtext?v=${videoId}&lang=en&fmt=srv3`
   - Wrap in try/catch (fetch failure = return null, silent fallthrough)
   - If response body is non-empty, contains `<`, and parsed text length > 50:
     ```javascript
     const xml = await res.text();
     if (xml && xml.trim().length > 0 && xml.includes('<')) {
       const text = parseTimedtextXml(xml);
       if (text.length > 50) {
         const result = { source: 'captions', text };
         cacheSet(cacheKey, result);
         return result;
       }
     }
     ```
   - Otherwise return `null` (caller handles fallback)

3. `transcriptHandler(req, res)` — async Express route handler for `GET /api/transcript/:videoId`:
   - Extract `videoId` from `req.params.videoId`
   - Call `fetchTranscript(videoId)`
   - If result is not null: `res.json({ videoId, source: result.source, text: result.text })`
   - If null, attempt description fallback:
     - Import `fetchVideoStats` from `./youtube.js`
     - Call `fetchVideoStats([videoId])` to get video data
     - If video exists and has `snippet.description` with length > 50:
       ```javascript
       const description = video.snippet.description;
       const result = { source: 'description', text: description };
       cacheSet(`transcript_${videoId}.json`, result);
       res.json({ videoId, source: 'description', text: description });
       ```
     - If neither available: `res.status(404).json({ error: 'NO_TRANSCRIPT', videoId, message: 'No transcript or description available' })`
   - Wrap entire handler in try/catch: on error, return 500 with `{ error: 'INTERNAL', videoId, message: error.message }`

Module uses CommonJS: `module.exports = { fetchTranscript, parseTimedtextXml, transcriptHandler }`.
